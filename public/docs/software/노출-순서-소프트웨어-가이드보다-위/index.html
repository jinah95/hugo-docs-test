## ⚙️ .env 설정

루트 디렉터리에 `.env.local` 파일을 생성하고 아래 서열대로 변수를 주입해 줍니다.

```env
# API 서버 주소
NEXT_PUBLIC_API_URL=[https://api.densflo.dev](https://api.densflo.dev)

# 개발 모드 디버깅 활성화 여부
NEXT_PUBLIC_DEBUG_MODE=true
