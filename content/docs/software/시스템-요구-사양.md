이 소프트웨어를 원활하게 구동하고 개발하기 위한 최소 및 권장 시스템 사양입니다.  
원활한 개발 프로세스 및 빌드 성능을 위해 가급적 **권장 사양 이상**의 환경을 권장합니다.

## 🖥️ 하드웨어 요구 사양

| 구분         | 최소 사양 (Minimum)                      | 권장 사양 (Recommended)                 |
| :----------- | :--------------------------------------- | :-------------------------------------- |
| **CPU**      | Intel Core i5 (8세대 이상) / AMD Ryzen 5 | Apple Silicon (M1 이상) / Intel Core i7 |
| **RAM**      | 8 GB                                     | **16 GB 이상**                          |
| **저장공간** | 5 GB 이상의 여유 공간 (HDD 가능)         | **20 GB 이상의 여유 공간 (SSD 필수)**   |
| **네트워크** | 패키지 다운로드를 위한 인터넷 연결       | 기가비트 이더넷 / Wi-Fi 6               |

> **💡 권장 사양 메모:**  
> 프론트엔드 환경에서 `pnpm` 빌드 및 로컬 테스트 서버를 동시에 구동하고, 백엔드 Docker 컨테이너까지 띄우려면 **RAM 16 GB 이상**이 정신 건강에 좋습니다. 8 GB 환경에서는 스와프 영역 공간을 충분히 확보해 주세요.

## 🛠️ 개발 툴 및 런타임 환경

원활한 프로젝트 구동을 위해 로컬 머신에 아래 런타임 버전을 맞춰주세요.

### 1. 런타임 (Node.js)

- **최소 사양**: `Node.js v18.17.0` (이하 버전에서는 일부 API 지원 불가)
- **권장 사양**: `Node.js v20.x` 혹은 `v22.x` (LTS 버전)

### 2. 패키지 매니저 (Package Manager)

- **필수 사양**: `pnpm v9.x` 이상 (`npm` 이나 `yarn` 빌드는 공식적으로 지원하지 않습니다.)

```bash
# 로컬 Node.js 및 pnpm 버전 확인용 커맨드
node -v
pnpm -v
```
